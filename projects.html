Class 01, Assignment 1 Assignment 1: Create an index.html file the given elements.

For this assignment, we had to create a simple webpage using the given tags. I created a table with a list of books I have and have not read with their title, author, and a one sentence review/expectation. Below the table I displayed each book's cover art.

Class 02, Assignment 2 Assignment 2: Create 2 bare bone webpages for a personal portfolio.

For this assignment, we had to create 2 basic pages for the start of our portfolio website. In these pages, I included as much info as I could (for the time being) about myself, how to reach me, and all the projects I have done. These pages follow the page layout (to the best of my abilities) of the mockups I created beforehand.

Class 03, Assignment 3 Assignment 3: Create a first-stylesheet.css

For this assignment, we had to replace all the basic deprecated HTML style tags used in assignment 2 and creat a first-stylesheet.css file. We had to keep in mind the original formatting of the pages created from assignment 2.

Class 04, Assignment 4 Assignment 4: Follow the mock-up

For this assignment, we were given a mock-up to and had to develop and design a webpage that looked similar to that mock up. In this assignment, we needed ot play around with positioning and floats to get the parts of the page to stay where they needed to be.

Class 05, Assignment 5 Assignment 5: Google Fonts, Font Awesome

For this assignment, we needed to incorporate Google Font and Font Awesome icons into our portfolio page. We needed to redesign of our portfolio site by adding an attractive color scheme and practicing our positioning skills.

Class 06, Assignment 6

For this assignment, we needed to update our portfolio site and add a new page to use at least three (3) of the following, in any combination on all pages: a CSS transition or transformation, :nth-style pseudo-selectors, FontAwesome icon, and/or Google Fonts embedded font. I added a contact page with various icons and transitions.
